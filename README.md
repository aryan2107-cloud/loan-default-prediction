# loan-default-prediction

Predicting loan defaults using machine learning.

## Problem

Banks need to know who will default before approving loans. This project builds a model that predicts default risk and explains its decisions.

## Data

Lending Club loan data — 30k+ loans with borrower info, loan details, and repayment outcomes.

## Approach

1. **EDA & Feature Engineering** — understand the data, create useful features
2. **Modeling** — compare classifiers, handle class imbalance, tune hyperparameters
3. **Interpretability** — SHAP explanations for model decisions
4. **Pipeline** — production-ready code that handles new data
